## Para iniciar el proyecto localmente

Clonar el repositorio

```bash
  git clone https://github.com/dani-assa/RollingBeer-backend.git
```

Instalar dependencias

```bash
  npm i
```

Iniciar el servidor

```bash
  npm run dev
```
